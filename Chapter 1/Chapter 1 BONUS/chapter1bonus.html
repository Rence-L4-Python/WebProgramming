HTML and CSS Resource Guide

HTML Tags

<html>

The html tag is used to contain all the other HTML elements, except the !DOCTYPE tag.


<!DOCTYPE html>
<html> 
<!--this is the element-->

    <head> 
    </head> 
    <body>
    </body>
</html> 
<!--closing tag-->

<head>

The head tag is used to contain metadata; data that can define styles, scripts, document title, and other meta tags.


<!DOCTYPE html>
<html>
    <head> 
<!--this is the element-->

    <style>
    p{
    }
    </style>
    </head> 
<!--closing tag-->

    <body>
    </body>
</html>
