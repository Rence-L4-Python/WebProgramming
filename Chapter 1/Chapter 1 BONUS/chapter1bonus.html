HTML Tags

<html>

The html tag is used to contain all the other HTML elements, except the !DOCTYPE tag.


<!DOCTYPE html>
<html> 
<!--this is the element-->

    <head> 
    </head> 
    <body>
    </body>
</html> 
<!--closing tag-->

<head>

The head tag is used to contain metadata -- data that can define styles, scripts, document title, and other meta tags.


<!DOCTYPE html>
<html>
    <head> 
<!--this is the element-->

    <style>
    p{
    }
    </style>
    </head> 
<!--closing tag-->

    <body>
    </body>
</html>

<body>

The body tag is used to contain all the contents of an HTML document. A few of examples of HTML tags are divs, headings, images, paragraphs, etc.


<!DOCTYPE html>
<html>
    <head> 
    </head> 
    <body> 
<!--this is the element-->

    </body> 
<!--closing tag-->

</html>

<div>

The div tag is a division in an HTML document that is used to contain other HTML elements. Div tags be styled with CSS and manipulated with JavaScript using HTML's class or id attributes.


<!DOCTYPE html>
<html>
    <head> 
    </head> 
    <body> 
        <div> 
<!--this is the element-->

        </div> 
<!--closing tag-->

    </body> 
</html>

<title>

The title tag is used to state the document's title. This tag can only be used with texts, and is shown in the browser's title bar as well as the page's tab.


<!DOCTYPE html>
<html>
    <head> 
        <title> 
<!--this is the element-->

        HTML and CSS Resource Guide
        </title> 
<!--closing tag-->

    </head> 
    <body> 
    </body> 
</html>

<!-->

The !-- tag is used to add comments in the source-code editor's viewport and in the page source. These are hidden from the browser and are primarily used to add helpful notes.


<!DOCTYPE html>
<html>
    <head> 
    </head> 
    <body> 
<!--comment anything-->

    </body> 
</html>

<p>

The p tag is used to state paragraphs and are used to add text to the page.


<!DOCTYPE html>
<html>
    <head> 
    </head> 
    <body> 
        <p> 
<!--this is the element-->

        Hello, world!
        </p> 
<!--closing tag-->

    </body> 
</html>

<a>

The a tag is used to create hyperlinks that allow a user to jump between sections or pages. The href attribute is used alongside this tag as the attribute indicates the link's destination.


<!DOCTYPE html>
<html>
    <head> 
    </head> 
    <body> 
        <a href="https://google.com"> 
<!--element-->

        Link to google
        </a> 
<!--closing tag-->

    </body> 
</html>

<sub>

The sub tag is used to create subscript texts, which are displayed half a character below the normal line, and in a smaller font than the default text.


<!DOCTYPE html>
<html>
    <head> 
    </head> 
    <body> 
        H
        <sub> 
<!--this is the element-->

        2
        </sub> 
<!--closing tag-->

        O
    </body> 
</html>

<img>

The img tag is used to embed images to an HTML page. This tag uses the src and alt attributes to function correctly. The first attribute is used to specify the path to the image, and the second attribute is used for describing an image if it fails to load.


<!DOCTYPE html>
<html>
    <head> 
    </head> 
    <body> 
        <img src="google.png" alt="Logo"> 
<!--element only-->

    </body> 
</html>

<hr>

The hr tag is used to add a horizontal rule which can be applied to define shifts of topic or a thematic break.


<!DOCTYPE html>
<html>
    <head> 
    </head> 
    <body> 
        <hr> 
<!--element only-->

    </body>
</html>

<pre>

The pre tag is used to display text "as-is" from the source code on the page. This tag preserves the line-breaks and spaces in the source code and is displayed in a fixed-width font.


<!DOCTYPE html>
<html>
    <head> 
    </head> 
    <body>
        <pre> 
<!--this is the element-->

Lorem ipsum dolor sit amet, consectetur
        <pre> 
<!--closing tag-->

    </body> 
</html>

<code>

The code tag is used to display computer code. Contents inside this tag will be displayed in the browser's default monospace font.


<!DOCTYPE html>
<html>
    <head> 
    </head> 
    <body> 
        <code> 
<!--this is the element-->

        html
        </code> 
<!--closing tag-->

    </body> 
</html>

CSS Properties

background-color

The background-color property is used to add background color to an element. An element's background includes its total size, padding, and border.


background-color: 
#222
;

color

The color property is used to change the color of text.


color: 
white
;

text-align

The text-align property is used to change the horizontal alignment of text in an element.


text-align: 
center
;

padding

The padding property is used to add extra space within an element. The space is added between the content and its border.


padding: 
20px
;

margin

The margin property is used to set the margins of an element. Negative values move the margin to its opposite side. This property is usually used with four values: top-right-bottom-left.


margin: 
20px 10px 5px 10px
;

font-size

The font-size property is used to change the size of a font.


font-size: 
36px
;

word-spacing

The word-spacing property is used to add or take away white space between words.


word-spacing: 
20px
;

line-height

The line-height property is used to change the height of a line. This is normally used on text elements.


line-height: 
20px
;
